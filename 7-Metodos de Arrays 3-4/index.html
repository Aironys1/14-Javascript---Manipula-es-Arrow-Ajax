Métodos de Arrays 3-4

Primeira forma, usando o map, ele mapea todo o array.

let lista = [45,4,9,16,25]; let lista2 = [];

Segunda forma

for(let i in lista){ lista2.push(lista[i] + 100); }

Usando o filter ele vai retornar true ou false.

lista2 = lista.filter(function(item){ if(item<30){ return true; }else{ return false; } });

O filter faz um filtro, podemos usar o if e else, para poder fitrar os item necessários.

Usando o every, ele fala para pegar todo mundo

lista2 = lista.every(function(item){ if(item<40){ return true; }else{ return false; } });

Mas irá dar falso, pois nem todos os item são menor que 40.

Usando a fução (some), significa alguns items

lista2 = lista.some(function(item){ if(item<40){ return true; }else{ return false; } });

Será verdadeiro na saída, pois alguns items são menor que 40.

Uma forma melhor para simplicar o return

return (item <40)? true : false;
