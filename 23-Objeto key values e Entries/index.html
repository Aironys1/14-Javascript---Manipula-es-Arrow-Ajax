23-Objeto key values e Entries

Object.keys, ele retorna todas as chaves que tem no obejto.

Object.values, ele nos retorna todos os valores do obejeto

Object.entries,ele gera um Array, dentro do Array eles nos geras os items

let lista = { nome:"Aironys", cargo: "Dev" }; console.log(typeof lista); let lista2 = ["Macarrão", "Feijão", "Pipoca"]; console.log(Object.entries(lista2)); let pessoa = { nome: "Amanda", idade:30, cargo: "Analista de Software", nacionalidade: "Brasileira" } console.log(Object.keys(pessoa));
