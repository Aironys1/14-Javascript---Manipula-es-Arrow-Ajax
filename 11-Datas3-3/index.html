Para trocar as datas, usamos a função d.setMonth(05)

let d = new Date(); let novoVAlor = d; d.setMonth(11)

Para ver as datas a partir de uma data de hoje

d.setDate(d.getDate() +5)

Daqui a 80 horas será:

d.setHours(d.getHours() +80)

Colocando um ano fixo de 1900

E aumentando, daqui á 80 anos

let d = new Date('1900'); let novoVAlor = d; d.setFullYear(d.getFullYear() +80)
